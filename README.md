
# Hardware Shop Inventory & Billing System

Desktop app for hardware shop stock and billing.

## Features
- Inventory: Add/view/update stock.
- Billing: Customer bills, full PDF + chit receipt (products, prices).
- Reports: Low stock.
- SQLite data.

## Run (one-time setup)
1. cd hardware_shop
2. pip install -r requirements.txt
3. python main.py

**Scripts:**
- Double-click `run.bat` (cmd).
- Right-click `run.ps1` > "Run with PowerShell".

Paths with spaces: use quotes.

## Usage
Inventory tab: manage items.
Billing: customer, add items, process → PDFs generated.

Sample data auto.
